# 🗺️ Roadmap — Mini RPG Games

> Last updated: 2026-06-22

---

## North Star 🎯

**Goal:** Build a portfolio of bite-sized RPG games that showcase AI-driven game development. Each game = 1 HTML file, playable anywhere, no install.

**Why it matters:**
- Portfolio piece for game dev / AI integration skills
- Playground for testing game mechanics
- Foundation for future multi-game platform

---

**Shipped:**
- [x] Forest Wanderer — top-down RPG, combat + NPC dialogue + slime AI
- [x] City Drift — GTA-lite, drive + missions + wanted level + police AI
- [x] GitHub Pages deployment
- [x] Project structure + docs
