## [1.0.0] — 2026-06-22

### 🎉 Initial Release

**Forest Wanderer v1.0**
- Top-down pixel RPG, 40×30 tile world
- 4-direction movement + dash + sword combat
- 3 NPCs with dialogue trees
- Slime AI with chase behavior
- HP/Lv/EXP/Gold progression
- Title screen + HUD + minimap (planned)

**City Drift v1.0**
- Top-down GTA-lite, 50×50 tile city
- 4 drivable vehicles (Sedan, Sports, Truck, Motor)
- Boost + brake + drift physics
- 25 NPCs + 40 parked cars + traffic AI
- 3 mission types: Taxi, Delivery, Chase
- 5-star wanted level + police AI + bust mechanic
- Real-time minimap with player/vehicles/police

**Infrastructure**
- GitHub Pages deployment (auto via Actions)
- Project structure: games/, docs/, assets/
- ROADMAP.md with 5-version plan
- MIT license
