# T-02.10: Touch controls + virtual buttons for mobile

## User request
"Cho thêm nút gạt tắt mở chế độ nút bấm ảo để test trên điện thoại!"

## Implementation
- **Toggle switch** 📱 Nút ảo (top center, click hoặc tap)
  - Desktop: opt-in via click
  - Touch device: auto-enable on first touch
- **Virtual joystick** (bottom-left)
  - Analog direction → W/A/S/D + arrow keys (with 0.3 dead zone)
  - Stick visual moves with touch position
- **7 action buttons** (bottom-right cluster):
  - E (lên/xuống xe) — large
  - F (bắn) — large
  - Space (đấm/phanh) — large, bottom-center
  - X (drift) — small
  - J (nhiệm vụ) — small
  - M (mute, red when active) — small
  - L (save) — small
- **Mouse fallback** for desktop testing without keyboard
- **Mobile responsive** (max-width: 600px breakpoint → smaller buttons)

## Architecture
Buttons wire to same `keys[]` and `justPressed[]` as keyboard handlers.
This means NO duplicate game logic — touch = keyboard at the JS level.
