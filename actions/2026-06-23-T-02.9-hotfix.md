# T-02.9: All-bug hotfix v1.2 — Actually tested with Puppeteer

## Trigger
User reported after v1.1 deploy:
- Một số nút không hoạt động
- Xe spawn chồng nhà → stuck
- Xe máy đi ngang
- Player xoay upside-down khi đi lên
- Không thấy hệ thống nhiệm vụ
- "Nên tự làm tự test thật!"

## Root Cause Analysis (via Puppeteer headless test)

### Bug #0 (CRITICAL — caused 90% of failures)
**Function `update(dt)` had no closing brace from line 912 to 1385.**
- `function update(dt)` opened at 912
- All subsequent code (476 lines) including `function startGame()` was nested INSIDE update()
- `startGame` was a local function declaration, not global
- `typeof startGame === 'undefined'` from puppeteer evaluation
- Click handlers registered inside `update` body — only ran when update was called
- But update returned early when !started, so listeners never registered
- → Game could not be started via click OR keyboard (Space, Enter both broken too)

### Bug #1: Player upside-down
`drawPlayer()` used `ctx.rotate(player.angle)` for whole body. When angle = -π/2 (walking up), entire sprite rotated 90° counter-clockwise.

**Fix:** Removed global rotation. Only gun barrel rotates. Body/head/arms/legs always UPRIGHT.

### Bug #2 + #4: Vehicles spawned in/on buildings
Old code: 4 vehicles at hardcoded positions — no collision check. 40 parked cars random but no neighbor check.

**Fix:** New `isPositionClear()` + `findClearSpawnPos()` with 1-tile radius.

### Bug #3: Bike facing horizontally
All vehicles spawned with `angle: 0` (pointing right) — bike w<h so long axis vertical but no rotation.

**Fix:** Spawn all with `angle: -π/2` (pointing up).

### Bug #5: Mission key M conflicted with mute
Both consumed `consume('m')`.

**Fix:** Mission trigger moved to J key.

### Bug #6: Mission panel showed "undefined"
`updateHUD()` types map only had 3/6 types.

**Fix:** Added race/escort/eliminate to types map.

## Lessons learned
1. NEVER trust static-only verification — `node --check` only validates JS syntax. Run actual browser test (puppeteer/playwright) for canvas games.
2. Nested function declarations hide globals — check function nesting before assuming global scope.
3. Function scoping saves hours of debugging — when something is `undefined` in browser but defined in source, check if nested.
4. Test actual user interaction — not just page.evaluate. Test clicks, keyboard, full game loop iteration.
5. Map collision checks need radius > 1 tile — center-point check passes when corner overlaps.
